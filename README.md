# Dynamic Form App

## Environments

```
1. Production

==> Branch Name: main

2. Development

==> Branch Name: develop

```

## Install

```sh

git clone https://github.com/M-Muneeb-Mughal/dynamic-form-app.git
cd dynamic-form-app
npm install
```

## Start development

```sh
npm run dev
```

## Build for production

```sh
npm run build
```

## Lint with

```sh
npm run lint
```
